Página de Prueba - Funcionalidad

Esta es una página de prueba para verificar que:

  • ✅ El clic derecho funciona (haz clic derecho en esta página)
  • ✅ La selección de texto funciona (selecciona este texto)
  • ✅ El navbar es clickeable
  • ✅ El botón hamburguesa funciona en móviles

Instrucciones de Prueba:

  1. Prueba el clic derecho: Haz clic derecho en cualquier parte de esta página
  2. Prueba la selección: Selecciona este texto con el mouse
  3. Prueba el navbar: Haz clic en los enlaces del menú superior
  4. Prueba responsive: Reduce el tamaño de la ventana y prueba el botón hamburguesa

Console del Navegador:

Abre las herramientas de desarrollador (F12) y revisa la consola para ver los mensajes de diagnóstico.
